Otto

Otto is a 3D platformer which I worked on for my major project of my course. Throughout the project the team learned a lot about developing a platformer and how much work and difficult it is to get the game feeling smooth. We learned that the movement had to feel smooth which ended up being the main focus of the mechanics. Overall I am happy with how the project ended up as we had about 4-5 months to come up with an idead pitch it and complete it as a functional game, we cut back on a couple fetaures from what we first planned as we did not want to over scope. Although some fetaures had been cut it did not take away from the main purpose of the project which was more so to be a demo of the platformer Otto which could be turned into a full game later on in the future if pursued.
